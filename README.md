# File Storage service

## Installation
1. _Install all dependencies:_
    ```bash
    $ yarn install
    ```
2. _Create an env file by following `.env.example` file:_
   >   `.development.env` **for development environment.**
   ---
   >   `.production.env` **for production environment.**

## Running the app

```bash
# development
$ make start-dev

# production
$ make start
```
